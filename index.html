Welcome!

This is a basic starter kit for Eleventy, with Forestry for editing and GitHub Pages for hosting baked in.

The site is largely based off the Eleventy starter site, which has been adapted and styled to be a simple basic blogging site. The aim is to make it as easy as possible for non-technical users to set up a blog using a static site generator. See the full instructions here.

Enjoy.
